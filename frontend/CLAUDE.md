# Commands
- `npm run dev`: Start Next.js development server.
- `npm run build`: Build for production.
- `npm run lint`: Run ESLint.

# Style Guidelines
- Use Next.js App Router (`src/app`).
- Keep components small and reusable in `src/components`.
- Differentiate clearly between Server Components (default) and Client Components (`"use client"`).
- Use CSS modules or standard CSS (`globals.css`), avoid inline styles for large blocks.
